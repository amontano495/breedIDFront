The PupperDex: It's like Shazam for dogs

PupperDex

The PupperDex is a web application that allows users to input an image of a dog, and it will output the breed of the given dog!

How it works:

The PupperDex opperates with HTML and CSS front end, and a python backend. An image is uploaded by the user and sent to our macine learning algorithm. This algorithm has been trained to recognize over 120 various breeds. After processing the image the algorithm returns the breed that most resembles the dog in the image.
